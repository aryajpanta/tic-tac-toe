# Tic-Tac-Toe

A modern, animated web-based Tic-Tac-Toe game with smooth interactions and responsive design.

## 🎮 Gameplay

Play the classic Tic-Tac-Toe game against a friend! Click on any empty cell to place your mark (X or O). The game automatically detects wins, ties, and keeps score across multiple rounds.

## 🎨 Features

- **Animated Interface**: Smooth hover effects, click animations, and winning line highlights
- **Responsive Design**: Works perfectly on desktop and mobile devices
- **Score Tracking**: Keeps track of wins for X, O, and ties
- **Modern Styling**: Gradient background and polished UI
- **Interactive Feedback**: Visual cues for every action

## 🚀 How to Play

1. Open `index.html` in your web browser
2. Click any empty cell to place your mark
3. X always goes first
4. The game will automatically detect when someone wins or if it's a tie
5. Use the buttons below the board to reset the game or reset scores

## 📱 Game Controls

- **Reset Game**: Clears the current board and starts a new game
- **Reset Score**: Clears all scores and starts fresh

## 🛠 Tech Stack

- **HTML5**: Semantic markup for the game interface
- **CSS3**: Modern styling with animations and transitions
- **Vanilla JavaScript**: Game logic without external dependencies

## 🎨 Animations

The game features several smooth animations:
- Hover ripple effects on cells
- Pop-in animations for X and O marks
- Winning line highlighting with pulse effects
- Scale animations on hover and click
